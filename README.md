# -Music-Player-in-JavaScript
**Music Player in JavaScript**
Welcome to the Music Player project repository! This project is a simple web-based music player built using HTML, CSS, and JavaScript. It allows users to play, pause, and control audio playback directly from their web browsers.

**Overview**
This project provides a basic music player interface that lets users play and control music files. The player includes features such as play/pause, volume control, and progress tracking. It’s a great example of how to create interactive web applications using JavaScript.

**Features**
Play/Pause: Controls to start and stop audio playback.
Volume Control: Slider to adjust the volume.
Progress Bar: Visual indicator of the audio playback progress.
Track Information: Displays the current track’s name and duration.
Playlist Support: Ability to load and switch between multiple tracks.
Responsive Design: Works on various devices and screen sizes.

**Usage**
Open index.html in your web browser.
Use the controls to play, pause, and adjust the volume of the music.
Interact with the progress bar to seek through the track.
To add your own music files:

Place your audio files in the assets directory (or wherever you choose).
Update the script.js file to include the path to your audio files in the playlist.
